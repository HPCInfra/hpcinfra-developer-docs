HPCInfra API Documentation

Welcome to the HPCInfra API documentation. Below are the available endpoints:

Authentication

Login

        POST /api/v1/auth/login
        Headers: { "Content-Type": "application/json" }
        Body:
        {
            "username": "your_email",
            "password": "your_password"
        }

Clusters

Get Cluster Details

        GET /api/v1/clusters/{cluster_id}
        Headers: { "Authorization": "Bearer YOUR_ACCESS_TOKEN" }

Jobs

Submit a Job

        POST /api/v1/jobs/submit
        Headers: { "Authorization": "Bearer YOUR_ACCESS_TOKEN" }
        Body:
        {
            "script": "bash_script.sh",
            "cluster_id": "12345"
        }

For more details, visit our www.hpcinfra.com
